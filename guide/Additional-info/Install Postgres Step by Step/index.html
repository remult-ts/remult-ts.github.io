# Install Postgres Step by Step

Download and install it from: Postgres download page

In this demo we've used Windows x86-64 version, 11.5

  1. Download and run the setup.
  2. Click next, next
  3. In the Select Components screen, un-check the Stack Builder we don't need it.

  1. Next, next
  2. You are prompted for a password, give it a password, and remember it (we'll need it later)

TIP

when creating this demo we used the password: MASTERKEY

From here on, just click next next next, till the setup is complete.
